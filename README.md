## 实验说明

​        本次实验实现了SVM算法，并且提供了两种实现方式，第一种是通过枚举参数的思想，不断趋近原数学模型的最优解。第二种是采用由约翰·普莱特于1998年发明的SMO算法来求解SVM对偶问题的解。在本次实验过程中我们从数学模型出发，采用python编程，从最朴素的求解方法到利用数学结论衍生出的优化算法，感受到了在机器学习算法中，数学思想对于算法优化有着举足轻重的作用。
